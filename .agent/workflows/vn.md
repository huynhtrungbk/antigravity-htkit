## Quy tắc

- Toàn bộ output bằng **tiếng Việt**
- Code comments bằng tiếng Anh
- Tài liệu trong `docs/` + `features/` bằng tiếng Việt

## Quy trình

1. Phân tích yêu cầu
2. Lưu phân tích vào `docs/XX-Analysis-Name-Date.md`
3. Tạo kế hoạch → lưu `docs/XX-Plan-Name-Date.md`
4. Gửi kế hoạch để duyệt
5. Sau khi duyệt → triển khai liên tục
6. Lưu báo cáo `docs/XX-Report-Name-Date.md`
