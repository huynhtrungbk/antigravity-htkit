## Available Workflows

| Command | Description |
|---------|-------------|
| `/ask` | Answer technical questions |
| `/plan` | Create implementation plan |
| `/fix` | Analyze and fix issues |
| `/debug` | Debug without auto-fixing |
| `/test` | Run tests and analyze |
| `/deploy` | Deploy to production |
