## Quick Start

```bash
# Copy kit to any project
cp -R .agent/ /path/to/your-project/.agent/
cp GEMINI.md /path/to/your-project/

# Check version
bash .agent/scripts/ht-version.sh show

# Get help
python3 .agent/scripts/ht-help.py
```
